
# Отладка надстроек Office на iPad и Mac

Visual Studio подходит для разработки и отладки надстроек в Windows, но с его помощью невозможно выполнять отладку надстроек на платформах iPad и Mac. Так как надстройки создаются с помощью кода HTML и Javascript, они рассчитаны на работу на многих платформах, но отрисовка HTML в разных браузерах может слегка отличаться. В этой статье описывается отладка надстроек на платформах iPad и Mac. 

## Отладка с помощью Vorlon.js 

Vorlon.js — это отладчик веб-страниц, похожий на средства, вызываемые клавишей F12. Он предназначен для удаленной работы и позволяет выполнять отладку веб-страниц на разных устройствах. Дополнительные сведения см. на [веб-сайте Vorlon](http://www.vorlonjs.com).  

Чтобы установить и настроить Vorlon: 

1.  Установите [Node.js](https://nodejs.org) и [Git](https://git-scm.com/) (если вы еще этого не сделали). 

2.  Установите Vorlon с помощью git и следующей команды: `git clone https://github.com/MicrosoftDX/Vorlonjs.git`.

3.  Установите зависимости с помощью команды `npm install`.

4.  Для надстроек требуется протокол HTTPS, поэтому во всех используемых ими сценариях, включая сценарий Vorlon, также должен использоваться протокол HTTPS. Следовательно, чтобы использовать Vorlon с надстройками, необходимо настроить Vorlon на применение SSL. В папке, выбранной для установки Vorlon, перейдите к папке /Server и отредактируйте файл config.json. Измените значение свойства **useSSL** на **true**. Там же можно включить подключаемый модуль для надстроек Office (изменив значение соответствующего свойства enabled на true). 

5.  Запустите сервер Vorlon с помощью команды `sudo vorlon`. 

6.  Откройте окно браузера и перейдите по адресу [http://localhost:1337](http://localhost:1337), где находится интерфейс Vorlon. Подтвердите доверие сертификату безопасности, когда вам будет предложено это сделать. Сертификат безопасности также можно найти в дочерней папке /Server/cert папки Vorlon. 

7.  Добавьте следующий тег скрипта в раздел `<head>` файла home.html (или основного HTML-файла) своей надстройки:
```    
<script src="https://localhost:1337/vorlon.js"></script>    
```  

Теперь при каждом запуске надстройки на устройстве она будет появляться в списке клиентов в Vorlon (в левой части интерфейса Vorlon). Вы можете удаленно выделять элементы DOM, выполнять команды и совершать многие другие действия.
